# Roadmap

## v0.3.0 验收版

- 扩展 Phaser 3 绑定面，覆盖动画、音频、相机、Tilemap、触控指针、场景控制、时间事件和 UI 文本样式。
- 保持 MoonBit 为主要实现语言，源码规模达到比赛参考要求。
- 提供可运行网页 Demo、MoonBit 测试、浏览器 smoke test、CI 和中文文档。
- 明确 v1 边界：专注 MoonBit JavaScript 后端和 Phaser 3，不扩散到其他游戏引擎或原生平台。

## v0.4.0 计划

- 增加更完整的多场景 Demo，包括开始菜单、暂停菜单、关卡切换和结算界面。
- 扩展浏览器 E2E 断言，覆盖动画播放、相机效果、触控滑动和 Tilemap 碰撞。
- 提供更多 Tiled JSON 示例地图和 spritesheet 示例素材。
- 补充包发布后的 Mooncakes 安装截图和版本变更记录。

## 长期方向

- 增加可选的 Phaser API 生成绑定，覆盖更多不常用但稳定的 Phaser 3 接口。
- 探索把 MoonBit 模型层导出给 JS 测试脚本，形成更强的跨语言验收描述。
- 收集社区游戏样例，沉淀 MoonBit Web 游戏开发最佳实践。
